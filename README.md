# TransConecta

Proyecto del SENA - Sistema web para asignación de servicios de transporte, gestión de rutas y control de horas de los conductores.

Este repositorio contendrá:

- Requisitos funcionales y no funcionales
- Diagramas UML (clases, casos de uso, secuencia, actividad)
- Diseño de la base de datos
- Desarrollo del frontend y backend
- Documentación técnica de la solución
